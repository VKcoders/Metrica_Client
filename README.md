## Running Locally

Clone the project

```bash
  git clone git@github.com:Vincenzofdg/Metrica.git
```

Navigate to the project directory

```bash
  cd Metrica
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
